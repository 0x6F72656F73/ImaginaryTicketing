# ImaginaryTicketing

[![Deploy][1]][2]
[![PRs Welcome][3]][4]
[![Discord chat][5]][6]

ImaginaryTicketing is a simple ticketing system for running CTF Competitions on discord. Be sure to checkout [ImaginaryCTF][7].

See [docs](docs/) for a usage guide.
